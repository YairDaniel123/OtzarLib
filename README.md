# OtzarLib
מאגר ספרים מותאמים לתוכנת אוצריא

מאגר זה מכיל קובצי TXT שעברו התאמה טכנית לשימוש בתוכנת אוצריא.

הקבצים מיועדים לשימוש פרטי בלבד ואינם חלק מההפצה הרשמית של התוכנה.
חלק מהתכנים הכלולים במאגר כפופים לזכויות יוצרים, ולכן אסורים בהפצה ציבורית, העתקה או שימוש מסחרי.

**מטרת המאגר**

 ריכוז וניהול קבצים מותאמים בפורמט אחיד
 
 שימוש אישי בסביבת אוצריא בלבד
 
 גיבוי וארגון תכנים פרטיים

**הבהרה משפטית**

העלאת הקבצים למאגר זה אינה מהווה ויתור על זכויות יוצרים ואינה מעניקה רישיון שימוש מעבר לשימוש פרטי אישי, בהתאם לדין.

האחריות לשימוש בתכנים חלה על המשתמש בלבד.
